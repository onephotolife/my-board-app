## ドメイン取得時の注意点

### 1. 取得前チェック項目
- [ ] Whoisで空き状況確認
- [ ] 商標データベース検索
- [ ] 類似サービスの有無確認
- [ ] SNSアカウント名の空き確認

### 2. 費用内訳の例
```
初年度：
- ドメイン登録料：1,500円
- Whois情報公開代行：無料～500円
- DNS設定：無料

更新費用（年間）：
- .com：1,500～2,500円
- .jp：3,000～4,000円
- .net：1,200～2,000円
```

### 3. 取得推奨レジストラ
- お名前.com
- ムームードメイン
- バリュードメイン
- エックスドメイン

### 4. SEO対策ポイント
- キーワードを含む（board, post, talk）
- ハイフンは最小限に
- 数字は意味があるもののみ
- 長すぎない（15文字以内）
